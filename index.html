RTK GPS Base Station System

This project represents the culmination of precision agriculture technology, implementing a complete RTK GPS base station infrastructure using ESP32-S3 microcontrollers paired with u-blox F9P high-precision GNSS modules. The system achieves centimeter-level accuracy across entire agricultural fields, enabling autonomous vehicles to navigate with unprecedented precision.
The architecture utilizes 4G cellular connectivity tunneled through Cloudflare to provide real-time RTK corrections to rovers in the field, eliminating the need for radio-based systems and their associated range limitations. This approach allows for scalable deployment across multiple fields and vehicles while maintaining robust, low-latency communication.
Key technical achievements include custom firmware development for ESP32 handling RTCM correction data streams, implementation of NTRIP protocols for correction distribution, and integration with existing agricultural automation systems. The base station provides continuous operation with automatic failover and monitoring capabilities.
Technologies: ESP32-S3, u-blox F9P, RTCM3, NTRIP, Cloudflare Tunnels, 4G LTE
Impact: Enabled centimeter-level autonomous navigation across 100+ acre deployments
Status: Production deployment at Flux Robotics

Precision Weed Spraying System

An advanced computer vision system for autonomous weed detection and targeted herbicide application, combining state-of-the-art object detection models with real-time agricultural robotics. The system uses GroundingDINO for open-vocabulary object detection paired with SAM2 (Segment Anything Model) for precise instance segmentation of individual plants.
Integrated with ROS (Robot Operating System) for coordinated control of the spraying mechanism, camera positioning, and vehicle navigation. The Fields2Cover library handles optimal path planning to ensure complete field coverage while minimizing overlaps and missed areas. Real-time processing runs on Jetson platforms with optimized GStreamer pipelines for minimal latency.
The system achieves 95%+ detection accuracy in varied field conditions, reducing herbicide usage by 70% compared to broadcast spraying while maintaining effective weed control. Custom algorithms handle challenging scenarios including overlapping plants, varying lighting conditions, and high-speed operation.
Technologies: GroundingDINO, SAM2, ROS, Fields2Cover, GStreamer, Jetson
Impact: 70% reduction in herbicide usage, sub-inch spraying accuracy
Status: Active development and field testing

Agricultural Path Planning

A sophisticated path planning system designed to overcome the inherent limitations of agricultural autopilot systems, specifically the Pixhawk's 400-waypoint restriction. This project developed novel solutions for uploading and managing thousands of waypoints required for complete field coverage in large-scale operations.
The solution employs a hybrid approach combining custom Lua scripting running on the Pixhawk itself with ground-side pymavlink Python applications for mission management. The system dynamically segments large missions into smaller chunks, uploads them progressively during operation, and seamlessly transitions between mission segments without interrupting field work.
Advanced algorithms handle complex field geometries, obstacle avoidance, headland turns, and optimal coverage patterns. Integration with the Fields2Cover library provides mathematically optimal path generation while custom extensions handle agricultural-specific constraints like row alignment, spray overlap management, and turn radius limitations.
Technologies: MAVLink, Pixhawk, Python (pymavlink), Lua, Fields2Cover
Impact: Enabled full-field autonomous coverage for 100+ acre operations
Status: Deployed in production autonomous systems
